# big-data-1
Εργασία που πραγματοποιήθηκε στα πλαίσια του μαθήματος «Big Data/Ανάλυση δεδομένων μεγάλου όγκου» στο Πανεπιστήμιο Μακεδονίας.

Στόχος είναι η ανάλυση δεδομένων καθυστερήσεων πτήσεων χρησιμοποιώντας το Apache Spark.
Θα αξιοποιηθεί το dataset flights_2000.csv, το οποίο περιλαμβάνει πληροφορίες για πτήσεις,
αεροδρόμια και καθυστερήσεις αναχωρήσεων. Κάθε εγγραφή περιέχει πεδία όπως: FL_DATE,
ORIGIN_AIRPORT, DEST_AIRPORT, DEP_DELAY, ARR_DELAY, CANCELLED, AIRLINE. Οι
φοιτητές καλούνται να υλοποιήσουν δύο διαφορετικές προσεγγίσεις ανάλυσης: με Spark RDD API και
Spark DataFrame API και να αξιολογήσουν επιδόσεις, εκφραστικότητα και αποδοτικότητα των δύο
προσεγγίσεων

Το αρχείο «flight_delay_spark_1.ipynb» περιέχει την υλοποίηση της προσέγγισης με Spark RDD API και το αρχείο «flight_delay_spark_2.ipynb» περιέχει την υλοποίηση της προσέγγισης με Spark DataFrame API. Το αρχείο «Αναφορά.pdf» περιέχει την ανάλυση των ευρημάτων που απαιτείται από την εργασία.
